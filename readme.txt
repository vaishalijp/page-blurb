This plugin helps to add meta title, meta excerpt and meta featured image to a page and also provides a shortcode to display this page meta as page blurb.

== Description ==
This plugin helps to add meta title, meta excerpt and meta featured image to a page. The meta data can be added on the add/edit page view in the dashboard.

The shortcode [display-pages-blurb] displays the page blurb of the current page child pages.

The shortcode [display-pages-blurb id=\"xx\"] displays the page blurb of the page with id \'xx\'.

== Installation ==
Search the plugin \"Fbm Page Blurb\" in the Plugins -> \'Add New\' screen on the dashboard.
Click the ‘Install Now’ button. 
Click on the \'Activate Plugin\' link to activate the plugin on your WordPress site. 


== Frequently Asked Questions ==
How do I add meta title, meta excerpt and alternate featured image to my page?
The Meta Fields can be added in the right section of the Add/Edit Page window in the dashboard. 

What is the shortcode to display the page blurb?
The shortcode is [display-pages-blurb]

What attributes does this shortcode accept?
The only attribute this shortcode accepts is page id. Example: [display-pages-blurb id=\"10\"]. This shortcode will display blurb of the page whose id is 10.

What if I dont provide an attribute?
The display-pages-blurb shortcode without any attribute will display the blurb of all child pages of the current page.
